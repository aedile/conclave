## Maintenance Notes

- This index is manually maintained. When adding a new document, add a row to the appropriate
  section.
- Run `find . -name '*.md' -not -path '*/.git/*' -not -path '*/.venv/*' -not -path '*/node_modules/*'`
  periodically to check for unindexed files.
- ADR status values: `Accepted`, `Superseded`, `Retired`, `Template`.
- A `Superseded` ADR has been replaced by a newer decision. The original file is retained but
  marked with a supersession notice at the top.
- A `Retired` ADR documented a decision that is no longer separately tracked. The file is retained.
- Phase backlog files are considered frozen (read-only) once the phase completes (✅ in BACKLOG.md).
